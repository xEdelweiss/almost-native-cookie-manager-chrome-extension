# Almost Native Cookie Manager

Chrome extension that adds new 'Cookies' tab in devtools.

### Implemented Features
 * List of cookies with details (Refresh)
 * Clear
 * Remove

### Planned Features
 * Add
 * Edit
 * Sort
 * Import / Export

### Bugs
 * Need refresh on tab update
